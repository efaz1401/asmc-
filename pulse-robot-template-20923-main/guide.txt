### Setting up Google Maps
1. Get Google Maps API key from Google Cloud Console
2. Add the API key to your environment variables
3. Update the Maps component in `src/components/ContactSection.tsx`
4. Replace the placeholder content (lines 147-156) with actual Google Maps embed

### Maps Integration Code Example
